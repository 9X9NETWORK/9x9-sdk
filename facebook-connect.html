Facebook Connect sample page

  • Facebook also needs a cross-domain file - channel.html
    Just copy and place it in your project then tell FB.init() where it is
    Of course, it should be place the same domain as your facebook app, in this example - dev.teltel.com
  • Once get an accessToken, write then back using api /api/users/{userId}/sns_auth/facebook
    After that, you can call this api to get more info or to check if the user is connected or not
